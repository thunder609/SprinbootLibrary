# SprinbootLibrary
## Spring Boot 
Proyecto SprinBoot Con Dependecias:

*  H2
*  Spring Data Jpa
*  Spring Web
*  Spring Boot Dev Tool
Aplicacion Api Rest Con acceso a base Datos H2 para persistir la informacion


El acceso se puede realizar desde Postman o Navegador...
